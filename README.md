## 🆘 Troubleshooting

### Blank Page?
1. Clear cache (`Ctrl + Shift + R`)
2. Check browser console (F12)
3. Verify all 3 files are uploaded
4. Wait 2 minutes for GitHub Pages to build

### Login Not Working?
1. Check `app-v2.js` is loaded
2. Open console and look for errors
3. Try another browser

### Pages Not Opening?
1. Verify `pages-functions.js` exists
2. Clear browser cache
3. Check console for errors

**Full troubleshooting guide:** [UPLOAD-GUIDE-AR.md](UPLOAD-GUIDE-AR.md)
