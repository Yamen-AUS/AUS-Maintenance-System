## 🚀 **كيف تستخدمه**

### **فتح الملف:**
```
1. double-click على الملف
2. سيفتح في Excel
3. سترى 4 تابس في الأسفل
```

### **التنقل:**
```
📊 Dashboard → للنظرة السريعة
📋 Tasks Data → للتفاصيل الكاملة
📈 Analytics → للتحليلات
📑 Summary → للملخص
```

### **إضافة مهام جديدة:**
```
1. اذهب إلى Tab: Tasks Data
2. اذهب لآخر صف (صف 33)
3. أضف البيانات الجديدة
4. الـ Dashboard و Analytics سيتحدثان تلقائياً
```
