## 🎯 الهدف
إنشاء ملف Excel احترافي لنظام الصيانة في **أقل من دقيقة واحدة**.

### ✅ تأكد من وجود الملفات التالية:

```
📁 المجلد الخاص بك/
  ├── 📄 generate_excel_professional.py     ← البرنامج
  ├── 📄 AUS_Maintenance_Data.json          ← البيانات

```

---

## 🚀 الخطوات (3 فقط!)

### 1️⃣ تثبيت المكتبة (مرة واحدة فقط)

افتح **Terminal** (Mac) أو **Command Prompt** (Windows) واكتب:

```bash
pip install xlsxwriter
```

⏱️ **الوقت:** 10-20 ثانية

---

### 2️⃣ تشغيل البرنامج

في نفس النافذة، اكتب:

```bash
python generate_excel_professional.py
```

**على Mac:**
```bash
python3 generate_excel_professional.py
```

⏱️ **الوقت:** 5-10 ثوانٍ

**ستظهر رسائل مثل:**
```
🚀 Starting AUS Maintenance Excel Generator...
📂 Loading maintenance data...
✅ Loaded 32 tasks
📝 Creating Excel file: AUS_Maintenance_Professional.xlsx
🎨 Creating cell formats...
📊 Creating Dashboard worksheet...
📝 Creating Tasks worksheet...
📊 Creating Analytics worksheet...
👥 Creating Vendors worksheet...
📄 Creating Executive Summary...

✅ SUCCESS! Excel file created: AUS_Maintenance_Professional.xlsx
```

---

### 3️⃣ فتح الملف

**الملف موجود في:** نفس المجلد الذي شغّلت منه البرنامج

**اسم الملف:** `AUS_Maintenance_Professional.xlsx`

**كيفية الفتح:**
- اضغط مرتين على الملف 🖱️
- أو اسحبه إلى Excel
- أو Right Click → Open With → Microsoft Excel

⏱️ **الوقت:** 2 ثانية

## 📤 رفع إلى OneDrive (اختياري)

### الطريقة السهلة:

1. افتح [onedrive.live.com](https://onedrive.live.com)
2. سجل الدخول
3. اضغط **Upload** → اختر الملف
4. اضغط **Share** → أضف بريد الفريق
5. اختر **Can Edit**
6. اضغط **Send**

⏱️ **الوقت:** دقيقة واحدة

## 🆘 مشاكل شائعة وحلول سريعة

### ❌ "Python is not recognized"
**الحل:** نزّل Python من [python.org](https://python.org) وحدد "Add to PATH"

### ❌ "No module named 'xlsxwriter'"
**الحل:** شغّل `pip install xlsxwriter` مرة أخرى

### ❌ "File not found: AUS_Maintenance_Data.json"
**الحل:** تأكد من وجود ملف JSON في نفس المجلد

### ❌ "Permission denied"
**الحل:** أغلق Excel إذا كان الملف مفتوحاً
