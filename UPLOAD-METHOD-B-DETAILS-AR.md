# 📋 تعليمات الرفع المباشر - الطريقة الأسهل

## ⚡ الطريقة الموصى بها: GitHub Web Editor

### الخطوات (3 دقائق فقط):

---

## 🔹 الخطوة 1: افتح GitHub

```
🔗 اذهب إلى: https://github.com/Yamen-AUS/AUS-Maintenance-System
```

---

## 🔹 الخطوة 2: احذف الملف القديم

1. اضغط على ملف **`app-v2.js`** من القائمة
2. اضغط على أيقونة **✏️ (Edit)** في الأعلى يمين الملف
3. ستفتح نافذة التحرير
4. اضغط على أيقونة **🗑️ (Delete)** بجانب Edit
5. أو: اختر الملف → اضغط Delete
6. اكتب رسالة التأكيد: `Delete old broken version`
7. اضغط **"Commit changes"** (الزر الأخضر)

---

## 🔹 الخطوة 3: أنشئ ملف جديد

1. ارجع للصفحة الرئيسية للمستودع
2. اضغط **"Add file"** (في الأعلى)
3. اختر **"Create new file"**
4. في خانة الاسم، اكتب: `app-v2.js`
5. **الآن ستحتاج المحتوى...**

---

### ✅ الحل الأمثل (اختر واحداً):

---

### ✨ الحل 1: Upload من جهازك (الأفضل)

**المطلوب**: يجب أن يكون لديك الملف `app-v2-FINAL.js` على جهازك

**الخطوات**:
1. في GitHub، اضغط **"Add file" → "Upload files"**
2. اسحب الملف `app-v2-FINAL.js` إلى المربع
3. أو اضغط **"choose your files"** واختر الملف
4. اكتب رسالة: `Upload fixed version`
5. اضغط **"Commit changes"**
6. **بعد الرفع**: 
   - اضغط على الملف
   - اضغط ✏️ Edit
   - غيّر الاسم من `app-v2-FINAL.js` إلى `app-v2.js`
   - احفظ

---

### 🔧 الحل 2: استخدم Git Clone (للمتقدمين)

إذا كان عندك Git على جهازك:

```bash
# 1. Clone هذا المشروع (الذي يحتوي على الملف المُصلح)
# (يجب أن أعطيك رابط المشروع الحالي)

# 2. انسخ الملف app-v2-FINAL.js

# 3. الصقه في مجلد AUS-Maintenance-System

# 4. ارفعه:
cd AUS-Maintenance-System
git rm app-v2.js
git add app-v2-FINAL.js
git mv app-v2-FINAL.js app-v2.js
git commit -m "Fixed all issues"
git push
```
