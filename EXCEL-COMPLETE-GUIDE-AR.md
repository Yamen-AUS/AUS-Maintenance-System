## 📤 خطوات النشر على OneDrive

### **خطوة بخطوة:**

```bash
1️⃣ تنزيل الملف
   • افتح AUS-Excel-Generator.html
   • اضغط "تنزيل ملف Excel الكامل"
   • الملف يُحفظ في Downloads

2️⃣ رفع على OneDrive
   • اذهب إلى onedrive.live.com
   • سجل الدخول
   • اضغط "Upload"
   • اختر الملف
   • انتظر الرفع (ثواني)

3️⃣ مشاركة مع الفريق
   • Right-click على الملف
   • اختر "Share"
   • أضف emails الفريق
   • حدد الصلاحيات:
     - Can edit (للمحررين)
     - Can view (للمشاهدين)
   • اضغط "Send"

4️⃣ العمل الجماعي
   • كل فرد يفتح الرابط
   • يمكن التعديل في نفس الوقت
   • التغييرات تُحفظ تلقائياً
   • لا حاجة للتنزيل/الرفع
```

