# 📖 كيف تفتح وتستخدم مولد Excel - شرح بسيط جداً

## 🎯 الطريقة الأسهل (3 خطوات)

### **الخطوة 1: تنزيل الملفات**
```
ستجد ملفين مهمين:
✅ AUS-Excel-Generator.html
✅ xlsx.full.min.js
```

### **الخطوة 2: فتح الملف**

#### **على Windows:**
```
1. اذهب إلى مجلد الملفات
2. ابحث عن: AUS-Excel-Generator.html
3. اضغط double-click عليه
4. سيفتح في المتصفح تلقائياً
```

#### **على Mac:**
```
1. اذهب إلى مجلد الملفات
2. ابحث عن: AUS-Excel-Generator.html
3. اضغط عليه مرتين
4. سيفتح في Safari أو Chrome
```

#### **إذا لم يفتح تلقائياً:**
```
1. اضغط Right-Click على الملف
2. اختر "Open with" أو "فتح باستخدام"
3. اختر:
   • Chrome
   • Firefox
   • Edge
   • Safari
```

### **الخطوة 3: تنزيل Excel**
```
بعد فتح الملف في المتصفح:

1. سترى صفحة جميلة باللون البنفسجي
2. سترى زرّين:
   
   🔵 "تنزيل ملف Excel الكامل"
   🔴 "تنزيل مع 4 أمثلة فقط"

3. اضغط على أي زر
4. انتظر ثواني
5. الملف سيُحمّل في Downloads
```

## 📁 أين أجد الملف بعد التنزيل؟

### **على Windows:**
```
1. افتح File Explorer
2. اذهب إلى: This PC > Downloads
3. ابحث عن:
   • AUS_Maintenance_Complete.xlsx
   أو
   • AUS_Maintenance_Sample.xlsx
```

### **على Mac:**
```
1. افتح Finder
2. اذهب إلى: Downloads
3. ستجد الملف هناك
```

## 🔄 كيف أرفعه على OneDrive؟

### **خطوات بسيطة:**

```
1️⃣ اذهب إلى: onedrive.live.com
   (في أي متصفح)

2️⃣ سجل الدخول بحسابك:
   • حساب Microsoft
   • أو Outlook
   • أو Hotmail

3️⃣ اضغط زر "Upload" أو "رفع"
   (في الأعلى)

4️⃣ اختر "Files" أو "ملفات"

5️⃣ ابحث عن ملف Excel في Downloads

6️⃣ اضغط "Open" أو "فتح"

7️⃣ انتظر الرفع (ثواني)

✅ تم! الملف الآن على OneDrive
```

## 👥 كيف أشارك الملف مع فريقي؟

### **بعد رفعه على OneDrive:**

```
1️⃣ اضغط على الملف (مرة واحدة)

2️⃣ اضغط زر "Share" أو "مشاركة"
   (في الأعلى)

3️⃣ أضف emails أعضاء الفريق:
   • اكتب email الشخص
   • اضغط Enter
   • كرر لكل شخص

4️⃣ اختر الصلاحيات:
   • Can edit (يمكنه التعديل) ✅ موصى به
   • Can view (المشاهدة فقط)

5️⃣ اضغط "Send" أو "إرسال"

✅ سيحصل كل شخص على رابط
```
