
**سيقوم تلقائياً بـ:**

2. ✅ تثبيت المكتبات المطلوبة (xlsxwriter)

4. ✅ فتح الملف تلقائياً

⏱️ **الوقت:** 20-30 ثانية

---

### 🍎 على Mac (الطريقة السهلة)

**1. افتح Terminal:**
```bash
cd ~/Desktop/AUS_Maintenance
```

**2. اجعل الملف قابل للتنفيذ:**
```bash
chmod +x run_generator.sh
```

**3. شغّل البرنامج:**
```bash
./run_generator.sh
```

⏱️ **الوقت:** 20-30 ثانية
