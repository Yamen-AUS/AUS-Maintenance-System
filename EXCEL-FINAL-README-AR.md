## 📤 رفع على OneDrive (3 خطوات)

### **الطريقة الكاملة:**

```bash
1️⃣ تنزيل الملف:
   • افتح AUS-Excel-Generator.html
   • اضغط الزر
   • انتظر ثواني
   • الملف في Downloads

2️⃣ رفع على OneDrive:
   • اذهب إلى: onedrive.live.com
   • سجل الدخول
   • اضغط "Upload"
   • اختر الملف
   • انتظر الرفع

3️⃣ مشاركة مع الفريق:
   • Right-click على الملف
   • اختر "Share"
   • أضف emails
   • حدد صلاحيات:
     - Can edit (تعديل)
     - Can view (عرض فقط)
   • اضغط "Send"
```
