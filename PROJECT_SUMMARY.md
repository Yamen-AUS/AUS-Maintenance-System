## 💡 How It Works

### **Technology Stack**:
```
Frontend: HTML5 + CSS3 + Vanilla JavaScript
Charts: Chart.js 4.4.0 (CDN)
Icons: Font Awesome 6.4.0 (CDN)
Fonts: Google Inter font (CDN)
Storage: Browser localStorage (5-10 MB capacity)
```

### **Architecture**:
```
Single-Page Application (SPA)
├── No server required
├── No database required
├── No installation required
└── Data stored locally in browser
```

### **Data Structure**:
```javascript
localStorage {
  "tasks": [20 task objects],
  "vendors": [12 vendor objects],
  "scheduleStatus": {recurring task statuses},
  "loggedIn": true/false
}
```
