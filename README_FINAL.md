## 🆘 حل المشاكل | Troubleshooting

### ❌ المشكلة: "Python is not recognized"
**الحل:** 
```
نزّل Python من: https://python.org
تأكد من تحديد "Add Python to PATH" أثناء التثبيت
```

### ❌ المشكلة: "No module named 'xlsxwriter'"
**الحل:**
```bash
pip install xlsxwriter
# أو
pip3 install xlsxwriter
```

### ❌ المشكلة: الملف لا يفتح
**الحل:**
```
- تأكد من وجود Microsoft Excel
- أو استخدم Excel Online
- Google Sheets قد لا يدعم جميع الميزات
```

### ❌ المشكلة: الرسوم البيانية لا تظهر
**الحل:**
```
افتح الملف في Microsoft Excel (ليس Google Sheets)
Excel Online يدعم الرسوم البيانية
```

**🔍 مشاكل أخرى؟** اقرأ **EXCEL_GUIDE_AR.md** القسم 🆘

## 📤 رفع إلى OneDrive | Upload to OneDrive

### الطريقة السريعة:

1. اذهب إلى [onedrive.live.com](https://onedrive.live.com)
2. سجل الدخول بحساب Microsoft
3. اضغط **Upload** → اختر الملف
4. اضغط **Share** → أضف بريد الفريق
5. اختر **Can Edit** للسماح بالتعديل
6. اضغط **Send**

**✅ جاهز للعمل الجماعي!**
