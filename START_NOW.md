## ✅ الخطوة الوحيدة

### على Windows (الأسهل):
```

```

```bash

# 2. اذهب إلى المجلد (مثلاً):
cd ~/Desktop/AUS_Maintenance

# 3. شغّل:
chmod +x run_generator.sh
./run_generator.sh
```

```bash
pip install xlsxwriter
```
